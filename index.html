📝 Conseils

  • 🔍 Vérifiez que votre question n'a pas déjà été posée.
  • 📝 Préférez un titre et une description rapide.
  • 🐛 Les demandes de fonctionnalités additionnelles ou les notifications de bogues devraient être ouvertes sur le traqueur de bogues approprie (i.e., le traqueur de bogue correspondant au projet, sur Bitbucket ou GitHub).
  • 📶 Des questions d'aide sur du code devraient être posée sur Stack Overflow et d'aide sur des maths sur Math Exchange.
  • 😊 Soyez gentil(le), civilisé(e) et poli(e) svp (comme toujours).
  • 😍 Si vous utilisez une emoji dans votre question, la réponse devrait arriver plus vite !
